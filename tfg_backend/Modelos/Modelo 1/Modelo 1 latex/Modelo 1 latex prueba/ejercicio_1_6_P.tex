\section*{Ejercicio 1.6 p}
Resolver la siguiente ecuación:
\[
-15X = -45
\]

\subsection*{Paso 1}
Aislamos la incógnita, teniendo en cuenta que el número que está multiplicando pasa dividiendo y no cambia el signo:
\[
X = \frac{-45}{-15}
\]

\subsection*{Paso 2}
Operamos:
\[
X = 3
\]

\[
X = 3
\]
